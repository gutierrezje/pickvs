# PickVs

## Development
pixi is used for Python package management for the backend, install that first if necessary. Then go into the directory, and install dependencies. 
```bash
cd backend
pixi install
```
Then to run a dev server

```bash
pixi run dev
```
Run tests with
```bash
pixi run test
```
Lint with Pyrefly and Ruff
```bash
pixi run lint
```
Format with Ruff
```bash
pixi run format
```
